Hexo博客搭建

发表于 2016-09-21

程序员如何便捷的搭一个有点逼格的博客(hexo+GitHub Pages)

工欲善其事，必先利其器

很早就一直想整理一个博客来给自己用，结果搞来搞去，自己就很懒惰，一直没有用心去做这个事，最近偶然看到stormzhang的博客，深有体会,便下定决心来搞一个博客,本文很简单，之前也有很多优秀的朋友们已经有讲解过了，我在这里就再详细说一下容易出错和不太好理解的地方吧。

我的博客使用 github pages+hexo来搭建，作为程序员，能用最大的同性交友网站，你不知道是多么的开心，本文一共分三部分，hexo的安装配置、github上面的配置、如何绑定域名。

1. hexo的安装与配置

Hexo是一个简单、快速、强大的静态博客框架,出自台湾大学生tommy351之手。下面是Hexo官网强调Hexo的四大特点：

  • 极速生成静态页面

  • 支持Markdown

  • 一键部署博客

  • 丰富的插件支持

安装前提
安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

  • Node.js
  • Git
    安装GIT
  • Windows：下载并安装 git.
  • Mac：使用 Homebrew, MacPorts 或下载 安装程序 安装。
  • Linux (Ubuntu, Debian)：sudo apt-get install git-core

  • Linux (Fedora, Red Hat, CentOS)：sudo yum install git-core
    安装 Node.js
    安装 Node.js 的最佳方式是使用 nvm
    cURL:
    $ curl https://raw.github.com/creationix/nvm/master/install.sh | sh
    Wget:
    $ wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh
    安装完成后，重启终端并执行下列命令即可安装 Node.js。
    $ nvm install stable
    安装Hexo

    如果是windows系统请打开Git bash，linux和mac直接在终端即可，新建目录hexo，进入到该目录:
    # 初始化hexo
    $ hexo init
    $ npm install
    $ hexo g # 或者hexo generate 生成静态网站
    $ hexo s # 或者hexo server，可以在http://localhost:4000/ 查看
    $ hexo deploy # 发布到服务器

2. github上面的配置

什么是GitHub Pages
GitHub Pages 本用于介绍托管在GitHub的项目，有300M免费空间，资料自己管理，保存可靠，用来做搭建一个博客再好不过了。每个帐号只能有一个仓库来存放个人主页，而且仓库的名字必须是username/username.github.io，这是特殊的命名约定。你可以通过http://username.github.io 来访问你的个人主页。
GitHub创建项目
在GitHub上面，在自己账户上创建一个名称为username.github.io的项目,username即为你的GitHub账户名称，一定要一致。
